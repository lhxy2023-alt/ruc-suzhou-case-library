# Draft Quality Checklist

Must pass before delivering draft:
- Hook: first 3-5 lines create clear curiosity or urgency.
- Coherence: each section supports one claim and transitions cleanly.
- Evidence: concrete examples/data where promise is strong.
- Style fit: diction and rhythm align with Style DNA card.
- WeChat readability: short paragraphs, scannable formatting.
- CTA: clear and specific next action.
- Format constraints: default 1-3 sentences per paragraph, digits for numbers.
- Symbol constraints: no em dashes.
- Source handling: reference facts and voice-transcript viewpoints are both represented.

Anti-patterns:
- Empty motivational cliches.
- Overuse of abstract nouns without examples.
- Repetitive sentence openings.

Fatal fail checks:
- Any hit in banned phrase list from `references/draft-dna-enforcement.md`.
- Any sentence using negation-frame templates such as `Not X, Y` or `Less X, more Y`.
- If fatal fail appears, rewrite and re-check before returning output.
