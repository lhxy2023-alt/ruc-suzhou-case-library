# Topic Evaluation Rubric

## 评分维度
每个选题角度按 100 分制评分：

1. 读者相关性 0-25
- 是否击中明确人群
- 是否对应真实痛点或欲望

2. 传播潜力 0-20
- 是否存在反差、争议、反直觉或情绪张力
- 是否适合公众号转发和收藏

3. 证据充足度 0-20
- 现有资料能否支撑主要论点
- 是否有案例、数据、原话、观察可用

4. 可写性 0-15
- 是否能自然展开成 1500-3000 字文章
- 是否能形成清晰结构

5. 与作者匹配度 0-10
- 是否适合作者视角和既有文风
- 是否有一手经验或真实判断

6. 业务目标匹配度 0-10
- 是否服务增长、转化、认知建立或关系维护

## 输出要求
每个角度必须同时给出：
- 一句话选题定义
- 目标读者
- 值得写的原因
- 主要风险
- 推荐级别
